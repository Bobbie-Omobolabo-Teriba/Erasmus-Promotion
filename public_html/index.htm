Important Information - PLEASE READ!

  • This is your student web directory. You can access it over FTP using details sent to you by email. If you put index.html in here, it will override this file (index.htm) and if you put a file index.php in the same directory, it will override ALL index files.

  • You can use PHP scripts to make connections to the MySQL database server (running on "localhost") or use any MySQL desktop client to access it as mysql01.comp.dkit.ie (port 3306). PHPMyAdmin is also available.

  • Connections to this server on ANY port can currently only be made from inside DkIT. If you want to work on files at home, export your MySQL database using PHPMyAdmin and import to XAMPP or similar on your laptop.

  • In all instances the password is the one mailed out to you, not your regular DkIT password (it can change regularly, inconvenient for PHP scripts with hardcoded connections).

  • Usernames are case sensitive. D123456 is not the same as d123456.

  • The web directories and databases are deleted and reset every summer. Keep backups!

  • The database password gods can only be appeased with coffee and pastries. You have been warned...

    • Example PHP Script

    <?php

    //Try this yourself by creating and uploading a PHP script to your FTP space

    $servername = "localhost";
    $username = "D000123456"; //substitute with YOUR username
    $password = "p@55!23"; //substitute with password sent to your email

    // Create connection
    $conn = new mysqli($servername, $username, $password); // Note how we are using mysqli(), not just mysql() *

    // Check connection
    if ($conn->connect_error) {
    die("Connection failed: " . $conn->connect_error);
    }
    echo "Connected successfully to database!";

    // Do something with the new connection....

    ?>

    * The MySQLi Extension ("MySQL Improved") is a relational database driver used in PHP programming to provide an interface with MySQL databases. A full list of MySQLi functions available to you on this server can be found here.
